cleanUp()method 
public static function cleanUp():void

Cycles through all the EventDispatcher objects and removes the ones that don't actually have the listener anymore. Consequently, if you use removeEventListener from EventDispatcher rather than here, the Dictionary in here will still think you haven't removed it. Calling cleanUp can fix this problem (unless you have the same event listener for multiple functions)
